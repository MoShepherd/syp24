UART RX Buffer für ein Bootloader Paket.

Dieser Buffer fasst ein gesamtes Bootloader Paket (3 Byte). Sobald der Buffer voll ist, signalisiert er dem Decoder, dass die Verarbeitung des empfangenen Paketes erfolgen kann.
