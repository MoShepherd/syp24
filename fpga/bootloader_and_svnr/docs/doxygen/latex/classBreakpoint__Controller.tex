\doxysubsection{AusfÃ¼hrliche Beschreibung}
Kontrolliert das Erzeugen/\+Löschen von Breakpoints, und überprüft ob ein Breakpoint im derzeitigen CPU Takt vorliegt. 

Die State-\/\+Machine dient zur Koordination von Erzeugung/\+Löschen der Breakpoints. Die Breakpoints werden in einem Array gespeichert, es können maximal 16 Breakpoints gleichzeitig bestehen. 0XFFFF beschreibt einen leeren/nicht aktivierten Breakpoint in der Liste. So kann ein Breakpoint bei bspw. Addresse {\itshape 0x0000} erzeugt werden. Die Addresse {\itshape 0x\+FFFF} spielt im SVNR keine Rolle, da dessen Addressraum nur {\itshape 1024} oder {\itshape 0x0400} Addressen umfasst. Der Index der ersten freuen Stelle des Arrays wird stets gespeichert, sodass Breakpoints an jeder Stelle des Arrays dynamisch gelöscht/hinzugefügt werden können.
