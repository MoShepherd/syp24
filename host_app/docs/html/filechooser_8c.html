void filechooser ( )

filechooser(): Öffnet bei Aufruf den Dateimanager des Host PCs

Der Dateimanager des Host-PCs öffnet sich im Home-Verzeichnis und man kann eine beliebige Datei auswählen, dessen Pfad dann dementsprechend (abhängig von der gewählten Option in der Applikation) anzeigt.

Noch zu erledigen
Evtl. implementieren, dass bei "Load Mnemonics" nur .asm-Dateien und bei "Load Binary" nur .bin-Dateien ausgewählt werden können.
