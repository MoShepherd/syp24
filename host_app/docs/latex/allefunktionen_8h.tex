\doxylink{debugger_8c_a19c86e0d31779f284e91d4d016448264}{debugger()}\+: Öffnet im Terminal ein neues Fenster mit Debugger Funktionen. 

Durch diese Funktion wird dem Nutzer die zuvor ausgewählte Datei angezeigt, mit welcher der Nutzer auch interagieren kann. Dem Nutzer werden Optionen angezeigt womit markierungen an den jeweiligen Stellen in der Textdatei auftauchen. Durch eine switch-\/case Anweisung werden verschiedene Optionen zur Interaktion mit dem Programmcode ermöglicht.
